# Лабораторная работа №8
## Тема: Синхронизация потоков в языке программирования python.
## Цель работы
приобретение навыков использования примитивов синхронизации в языке программирования Python версии 3.x.
## Индивидуальное задание
C использованием многопоточности для заданного значения x необходимо организовать конвейер, в котором сначала в отдельном потоке вычисляется значение первой функции, после чего результат вычисления должны передаваться второй функции, вычисленной в отдельном потоке. Потоки для вычисления значений двух функций должны запускаться одновременно. Значения сумм рядов должны быть вычислены с тонностью ε=10^(-7).
